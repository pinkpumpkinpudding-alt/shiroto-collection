## 🎯 プロジェクト概要

### サイトの目的
- 成人向け作品の中でも「素人」「新人」「初々しい雰囲気」を好むユーザー向けの特化型まとめサイト
- FANZAアフィリエイトプログラムと連携し、作品の魅力を分かりやすく紹介
- 清潔感があり、見やすく、売り込み感を抑えたデザイン
- ターゲット: 20代〜40代男性
- 最終目的: FANZAアフィリエイトの成約率向上

### デザインコンセプト
- **清潔感重視**: ゴテゴテしたアダルト感を排除
- **見やすさ優先**: カード型デザイン、余白多め
- **抜け感**: 白・淡いグレー・くすみ系アクセントカラー
- **スマホファースト**: レスポンシブデザイン完全対応

### 絶対条件（コンプライアンス）
✅ 掲載対象はすべて成人向け合法コンテンツのみ  
✅ 出演者は全員18歳以上の成人  
✅ 未成年・学生・児童に見える表現は禁止  
✅ 18歳未満利用禁止の注意文を複数箇所に配置  
✅ 違法性を感じさせる表現は禁止

## 🛠 カスタマイズガイド

### 作品を追加する方法

`script.js` の `productsData` 配列に新しいオブジェクトを追加:

```javascript
const productsData = [
    // ... 既存の作品 ...
    {
        id: 16,                           // ユニークなID（連番推奨）
        title: '新作タイトル',            // 作品タイトル
        genre: 'amateur',                 // ジャンル（下記参照）
        tags: ['素人', 'タグ2', 'タグ3'], // タグ配列
        description: '一覧用の短い説明',  // 2行程度
        longDescription: '詳細ページ用の説明', // 詳細な説明
        ranking: 16,                      // ランキング順位
        isNew: true                       // 新着バッジ表示（true/false）
    }
];
```

**ジャンルキー一覧:**
- `amateur` - 素人系
- `newcomer` - 新人・初撮り
- `clean` - 清楚・ナチュラル
- `documentary` - ドキュメンタリー風
- `daily` - 日常系
- `couple` - カップル・彼女系
- `natural` - ナチュラル系

### 新しいジャンルを追加する方法

1. **script.js のジャンルマップに追加:**
   ```javascript
   const genreMap = {
       // ... 既存のジャンル ...
       'new-genre': { 
           name: '新ジャンル名', 
           description: 'ジャンルの説明' 
       }
   };
   ```

2. **index.html のジャンルナビゲーションに追加:**
   ```html
   <div class="genre-card" onclick="navigateTo('genres', 'new-genre')">
       <div class="genre-icon">🆕</div>
       <h3 class="genre-title">新ジャンル名</h3>
       <p class="genre-desc">ジャンルの説明</p>
   </div>
   ```

### カラーテーマの変更

`styles.css` の CSS変数を編集:

```css
:root {

    /* アクセントカラー */
```

## 🐛 トラブルシューティング

### 年齢確認モーダルが再表示される
→ `sessionStorage` が有効か確認。プライベートブラウジングでは毎回表示されます。

### ハンバーガーメニューが動作しない
→ `script.js` が正しく読み込まれているか確認。コンソールエラーをチェック。

### 作品が表示されない
→ `productsData` 配列が正しく定義されているか確認。

### スタイルが崩れる
→ `styles.css` のパスが正しいか確認。ブラウザキャッシュをクリア。

### FANZAリンクが機能しない
→ `AFFILIATE_ID` が正しく設定されているか確認。
